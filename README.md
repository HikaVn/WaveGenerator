# WaveGen — 共振サウンドジェネレーター

スマホのブラウザだけで動く、スワイプ操作の音波発信器（PWA）です。
バイオリンなどの楽器の **共振点（レゾナンス）探索** を主な用途に設計しています。
インストール不要・オフライン動作・ホーム画面追加でネイティブアプリのように使えます。

## 主な機能

- **スワイプで周波数を操作** — パッドを横スワイプで周波数（対数＝音楽的）、縦スワイプで音量。
- **柔軟なレンジ** — 任意の Hz レンジを設定。バイオリン各弦・共振探索・全帯域などのプリセット付き。
- **共振マーク & プロファイル記憶** — 見つけた共振点を周波数・音名・メモ付きで保存。楽器ごとにプロファイル管理（localStorage）。エクスポート/インポート対応。
- **マイクによる共振測定** — 発信音に対する楽器の応答を FFT で測定。スイープでレスポンスカーブを記録し、**ピークを共振点として自動検出**。
- **オートスイープ** — 指定レンジを対数スイープし、応答カーブを描画。
- **シンプルなシンセサイザー** — サイン/三角/ノコギリ/矩形＋加算合成（オルガン/ストリングス/ホロウ）、ビブラート、トレモロ、うなり検出用デチューン、A4 基準ピッチ調整、音階スナップ、セント単位の微調整。

## 操作

| 操作 | 動作 |
| --- | --- |
| 横スワイプ | 周波数を上下（対数） |
| 縦スワイプ | 音量 |
| ダブルタップ | 再生 / 停止 |
| 長押し | 現在の周波数を共振マークに記録 |
| 微調整トグル | スワイプ感度を下げて細かく調整 |
| 音階スナップ | 平均律の半音にスナップ |

## 共振点の調べ方

1. 「スイープ & 共振測定」パネルで **マイクをON**（AGC/ノイズ抑制は無効化して生の応答を取得）。
2. レンジを楽器の探索帯域に設定（例: バイオリン本体共振なら 160–800 Hz）。
3. 楽器の近くで **スイープ開始**。発信音が共振周波数に達すると楽器本体が強く鳴り、応答カーブにピークが出ます。
4. **ピーク検出してマーク** で共振点を自動抽出・保存。

> パッド上の緑のバンドはバイオリンの代表的な共振域（A0/本体）の目安、橙のティックは保存済みマークです。

## 使い方（ホスティング）

静的ファイルのみなので、任意の静的サーバーで配信できます。

```sh
# 例: ローカルで確認
python3 -m http.server 8000
# → スマホで http://<PCのIP>:8000/ を開く
```

マイク機能は **HTTPS（または localhost）** が必要です。`getUserMedia` の制約です。

## 技術

- Web Audio API（OscillatorNode / PeriodicWave / AnalyserNode）
- 純粋な ES Modules・依存ライブラリなし
- Service Worker によるオフラインキャッシュ、Web App Manifest による「ホーム画面に追加」

```
js/analyzer.js          マイク入力・FFT・ピーク検出
js/gestures.js          スワイプ/タップ操作
js/notes.js             周波数↔音名変換
js/storage.js           マーク・プロファイル永続化
manifest.webmanifest    PWA マニフェスト

```
